For each project: what I couldn't change, and what I could. That gap is where the design lives.

Locked — given, inherited, or required
Unlocked — open to design
01 Customer Data Platform — Segment & Condition Builder Case study →

Segment builder for marketing teams using boolean logic

Locked
  • AND/OR/NOT logic must be preserved in full
  • Existing data attributes could not be renamed or reorganized
  • Must run in a React SPA with existing API contracts
  • Segment preview must show live count before saving
Unlocked
  • How conditions are visually represented and indented
  • The autocomplete search UX for attribute selection
  • How group nesting was communicated spatially
  • Error and validation state design

The logic was non-negotiable — but how that logic appeared on screen was entirely open. Making the boolean tree readable was the design problem.

02 3BB Member App & Website Case study →

Migration of live product used by 1M+ users

Locked
  • The new backend system's data model and fields
  • Feature scope: only migration-affected flows
  • App was live in production throughout
  • Existing design system patterns and components
Unlocked
  • How users were guided through new account linking
  • Error states for edge cases the old system didn't have
  • Copy and tone for the migration messaging
  • Which screens needed redesign vs only copy updates

The scope was tightly locked. The design freedom was in how to make the migration invisible — transitions that felt like nothing changed, even when everything underneath had.

03 Counter Service POS Terminal Proposal →

POS terminal for cashiers processing at speed

Locked
  • All existing element positions (muscle memory)
  • Touch target areas and minimum sizes
  • Hardware: fixed screen resolution and pixel density
  • No new features — no new flows
Unlocked
  • Typography weight and size hierarchy
  • Color contrast and visual clarity
  • Grouping by proximity without moving elements
  • State differentiation (active vs inactive items)

Nothing could move. Every gain in clarity had to come from within the existing spatial footprint — through weight, contrast, and grouping alone.
